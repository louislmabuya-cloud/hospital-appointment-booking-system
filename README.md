# Hospital Appointment Booking System

## Overview
This project is a Hospital Appointment Booking System designed for HealthyCare Medical Group to replace manual appointment scheduling with a centralized digital solution.

## Problem
The current manual system causes delays, double bookings, and inefficient scheduling.

## Solution
A web-based system that allows patients to:
- Register and log in
- Search doctors
- Book appointments in real time
- Receive SMS/email notifications
- Manage bookings

## Key Features
- Patient registration & authentication
- Doctor search by speciality and availability
- Real-time appointment scheduling
- Automated reminders
- Admin dashboard with reporting
